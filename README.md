# Bin Picking Robot

Vision-guided SCARA robot for automated pick-and-place of poker chips using an overhead camera and suction or claw end effector.

## Why this project

This project is designed to mimic real industrial pick-and-place automation systems rather than a generic hobby robot arm. A SCARA-style robot was chosen because it is mechanically simple, fast, compact, and commonly used in manufacturing automation for horizontal pick-and-place tasks.

## Mechanical Architecture

- SCARA-style robot arm
- Overhead camera for top-down object detection
- Rotational joints for planar positioning
- Vertical end effector motion for picking
- Suction or claw-based pickup mechanism
